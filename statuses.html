To add a status when writing a Wiki page, write {% raw %}{% include status.html id=number %}{% endraw %}.

Example: {% raw %}{% include status.html id=1 %}{% endraw %} will becomes {% include status.html id=1 %} when rendered.

Example: {% raw %}{% include status-icon.html id=20 %}{% endraw %} will becomes {% include status-icon.html id=20 %} when rendered.
