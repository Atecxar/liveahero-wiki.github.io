To add item when writing Wiki page, write {% raw %}{% include item.html id=number %}{% endraw %}.

Example: {% raw %}{% include item.html id=2 %}{% endraw %} will becomes {% include item.html id=2 %} when rendered.

Example: {% raw %}{% include item-icon.html id=42 %}{% endraw %} will becomes {% include item-icon.html id=42 %} when rendered.
